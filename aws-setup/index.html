Working with AWS in Ambit

Ambit integrates with Amazon Simple Storage Service (Amazon S3) to help centralize various files into a cloud storage that can then be re-imported easily from any computer.

Setup

To successfully utilize the S3 bucket, AWS credentials need to be configured in your environment for a role that has access to the S3 bucket. Follow this guide for step-by-step instructions.

Configure AWS Settings

Once the account has the correct permissions, find the AWS Settings menu inside of AWS Ambit Scenario Mode, then select the AWS Region of your S3 bucket and input the bucket name.

Note: S3 bucket names must adhere to the S3 Bucket Naming Rules

Ambit will utilize this setting throughout to upload and download from the bucket specified. Ambit will automatically create a S3 bucket if one does not exist when uploading.
